# HR-Analytics-Dashboard
This Power BI project analyzes HR data to monitor employee performance, attrition trends, and level jobs by employee

The dashboard helps HR teams to:

1-Track active worker and retreachment

2- employees who are  due for promotion and not due

3- Total employees divided by gender ( Male, Female)

4- Due retrenchment and due promotion by department


📊 Dataset Details


Source: Kaggle (HR Analytics Dataset)

Domain: Human Resources / Employee Analytics

Data Model Type: Star Schema (Fact Table + Dimension Tables)


🎯 Problem Statement


HR management needs insights into:

- percentage of Hight Rating and low Rating Employees
- Total employees and due promotion per job role
- Total employee by level job

⚙️ Steps I Performed

  
1- Data Cleaning (Power Query)

   Standardized data types

   Handled missing values

   Renamed columns for clarity

2- Data Modeling

   Created one-to-one relationships

   Centralized HR Analytics Data as the fact table

   Linked HR employee data as dimension

Microsoft Power BI Desktop
